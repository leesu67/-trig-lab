⚡ 快速選擇三角形
📝 輸入三邊長 (a, b, c)

李向宇老師的破解心法：

  1. 已知兩邊夾角 → 用 ½bc sin A (最常用！)
  2. 已知三邊長 → 用海龍公式 √[s(s-a)(s-b)(s-c)]
  3. 已知內切圓 → 用 rs (幾何切割法)
  4. 已知外接圓 → 用 abc/(4R)
